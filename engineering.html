Op-Amps & Comparators

The Two Golden Rules

Rule 1: V+ = V− (virtual short)
Rule 2: I+ = I− = 0 (no input current)

Only apply when the op-amp is in its linear region (negative feedback present). In comparator mode (open-loop or positive feedback), the virtual short does NOT apply.

Key Insight

Negative feedback → use virtual short. Positive feedback or open-loop → comparator, output rails to ±Vsat.

ECE 371Jaeger & Blalock Ch. 10

Inverting

Av = −Rf / R1   |   Rin = R1

Non-Inverting

Av = 1 + Rf / R1   |   Rin ≈ ∞

Non-inverting always has gain ≥ 1. Inverting can attenuate (|gain| < 1) or amplify.

An op-amp run open-loop. Compares V+ and V−:

V+ > V− → Vout = +Vsat
V+ < V− → Vout = −Vsat (or 0V single supply)

LM311 is open-collector — needs a pull-up resistor to define HIGH.

Schmitt Triggers & Hysteresis

A comparator with memory. Uses positive feedback to create two switching thresholds (VTH and VTL), preventing chatter from noisy signals.

Hysteresis = VTH − VTL
Key Insight

Think of it like a light switch with a dead zone — you have to flip it decisively, not nudge it.

Vp = Vin × (Rf / (Rin + Rf)) + Vout × (Rin / (Rin + Rf))

Set Vp = Vref and solve for Vin with Vout = +Vsat (gives VTL) and Vout = −Vsat (gives VTH).

555 Timers

Upper threshold (Pin 6): 2/3 × VCC → triggers DISCHARGE
Lower threshold (Pin 2): 1/3 × VCC → triggers CHARGE

Capacitor bounces between 1/3 and 2/3 of VCC, toggling output each time.

tHIGH = 0.693 × (RA + RB) × C
tLOW = 0.693 × RB × C
f = 1.44 / ((RA + 2RB) × C)
Duty Cycle = (RA + RB) / (RA + 2RB)
Key Insight

Duty cycle always > 50% in standard astable — charge path includes RA + RB, discharge only RB.

ECE 371Jaeger Ch. 14
T = 1.1 × R × C   (precisely: T = RC × ln(3) ≈ 1.0986 × RC)

One-shot. Triggered by falling edge on Pin 2. Output goes HIGH for duration T, then returns LOW.

Wien-Bridge Oscillators

|Aβ| ≥ 1 (loop gain ≥ 1 at startup)
∠Aβ = 0° (total phase = 0°)

Wien-Bridge: β = 1/3 at fosc → need A ≥ 3 → Rf/R1 ≥ 2
fosc = 1 / (2πRC)
Key Insight

At startup loop gain > 1 to grow oscillations. Diode stabilization brings it to exactly 1 for steady state.

Small signal → diodes off → gain > 3. Large signal → diodes conduct → rd parallels Rf → gain drops to 3.

Effective Rf = Rf ∥ rd
A = 1 + (Rf ∥ rd) / R1 → 3

Transformers

Determines core branch — Rc and Xm.

Setup: Energize LV at rated voltage, HV open.
Measure: Voc, Ioc, Poc

cos(φ₀) = Poc / (Voc × Ioc)
Rc = Voc² / Poc
Im = √(Ioc² − (Poc/Voc)²)
Xm = Voc / Im

Determines series branch — Req and Xeq.

Setup: Short LV, reduced voltage on HV until rated current.
Measure: Vsc, Isc, Psc

Zeq = Vsc / Isc
Req = Psc / Isc²
Xeq = √(Zeq² − Req²)
Key Insight

OC test → core losses (iron). SC test → copper losses (I²R). They separate the two loss mechanisms.
