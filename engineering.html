Op-Amps & Comparators

The Two Golden Rules

Rule 1: V+ = V− (virtual short)
Rule 2: I+ = I− = 0 (no input current)

Only apply in the linear region (negative feedback). Comparator mode (open-loop / positive feedback) → no virtual short, output rails to ±Vsat.

Key Insight

Negative feedback → virtual short. Positive feedback → comparator.

ECE 371Jaeger & Blalock Ch. 10

Inverting

Av = −Rf / R1   |   Rin = R1

Non-Inverting

Av = 1 + Rf / R1   |   Rin ≈ ∞

Non-inverting always has gain ≥ 1. Inverting can attenuate or amplify.

Op-amp run open-loop. Compares V+ and V−:

V+ > V− → Vout = +Vsat
V+ < V− → Vout = −Vsat (or 0V single supply)

LM311 is open-collector — needs a pull-up resistor for HIGH.

Schmitt Triggers & Hysteresis

A comparator with memory. Positive feedback creates two thresholds (VTH and VTL), preventing chatter from noise.

Hysteresis = V_TH − V_TL
Key Insight

Like a light switch with a dead zone — flip decisively, not nudge.

Vp = Vin × (Rf / (Rin + Rf)) + Vout × (Rin / (Rin + Rf))

Set Vp = Vref, solve for Vin with Vout = +Vsat (→ VTL) and Vout = −Vsat (→ VTH).

555 Timers

Upper (Pin 6): 2/3 × Vcc → DISCHARGE
Lower (Pin 2): 1/3 × Vcc → CHARGE

Capacitor bounces between 1/3 and 2/3 Vcc, toggling output each threshold hit.

t_HIGH = 0.693 × (RA + RB) × C
t_LOW = 0.693 × RB × C
f = 1.44 / ((RA + 2RB) × C)
Duty = (RA + RB) / (RA + 2RB)
Key Insight

Duty > 50% always in standard astable — charge goes through RA+RB, discharge only RB.

ECE 371Jaeger Ch. 14
T = 1.1 × R × C   (precise: T = RC × ln(3))

One-shot. Falling edge on Pin 2 triggers. Output HIGH for T, then LOW.

Wien-Bridge Oscillators

|Aβ| ≥ 1 at startup   |   ∠Aβ = 0°

β = 1/3 → A ≥ 3 → Rf/R1 ≥ 2
f_osc = 1 / (2πRC)
Key Insight

Startup: gain > 1 to grow. Diodes stabilize to exactly 1 for steady state.

Small signal → diodes off → gain > 3. Large signal → diodes on → rd ∥ Rf → gain → 3.

Effective Rf = Rf ∥ rd
A = 1 + (Rf ∥ rd) / R1 → 3

Transformers

Determines core branch — Rc and Xm.

Setup: Energize LV at rated V, HV open
Measure: Voc, Ioc, Poc

cos(φ₀) = Poc / (Voc × Ioc)
Rc = Voc² / Poc
Im = √(Ioc² − (Poc/Voc)²)
Xm = Voc / Im

Determines series branch — Req and Xeq.

Setup: Short LV, reduced V on HV until rated I
Measure: Vsc, Isc, Psc

Zeq = Vsc / Isc
Req = Psc / Isc²
Xeq = √(Zeq² − Req²)
Key Insight

OC → core/iron losses. SC → copper/I²R losses. Two tests separate two loss mechanisms.
