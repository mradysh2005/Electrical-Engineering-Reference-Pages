BJT & MOSFET
Key Parameters

Every critical parameter, formula, and insight you need — from physics to small-signal models.

BJT Core DC Parameters
β (hFE)
Current Gain
(common emitter)
β = IC / IB

Ratio of collector to base current. Typically 50–300 but highly variable. Never design around a specific β value — it drifts with temperature, IC, and device variation.

α
Common-Base
Current Gain
α = IC / IE = β/(β+1)

Always less than 1 (typically 0.99+). Relates to β by α = β/(β+1). Tells you how much of IE actually makes it to the collector. α → 1 as β → ∞.

IS
Saturation
Current
IC = IS · e^(VBE/VT)

Extremely small (~10⁻¹⁵ A). Sets the absolute scale of IC. The exponential relationship means every 60mV increase in VBE multiplies IC by 10.

VT
Thermal
Voltage
VT = kT/q ≈ 26mV @ 300K

k = Boltzmann constant, T = temp in Kelvin, q = electron charge. This appears everywhere in BJT analysis. Increases ~0.086 mV/°C — critical for tempco calculations.

VA
Early
Voltage
IC = IS·e^(VBE/VT)·(1 + VCE/VA)

Models base-width modulation (IC rising with VCE in active region). Typically 50–200V. Higher VA = better current source behavior. Defines the finite output resistance ro.

VBE ≈ 0.6–0.7V for Si BJT. It DECREASES ~2mV/°C — this is why BJT amplifiers drift with temperature.
MOSFET Core DC Parameters
VTH
Threshold
Voltage
VGS > VTH to turn on

The gate-source voltage needed to invert the channel. Typically 0.4–1V for NMOS. Decreases with temperature (~−2mV/°C). Affected by body effect if VS ≠ 0.

k'n / μnCox
Process
Transconductance
k'n = μn · Cox

Combines carrier mobility (μn) and gate oxide capacitance per area (Cox). Set by the fabrication process — you cannot change it. μn ≈ 2–3× μp, so NMOS is naturally faster/stronger.

W/L
Aspect
Ratio
kn = k'n · (W/L)

The designer's primary knob. Wider W → more current drive. Shorter L → faster but more λ (channel length modulation). kn is the device transconductance parameter.

ID (sat)
Drain Current
Saturation
ID = (kn/2)(VGS − VTH)²

The square-law relationship. VGS−VTH is called overdrive voltage (VOV). Squaring means doubling VOV quadruples current. Valid when VDS ≥ VGS − VTH.

λ
Channel Length
Modulation
ID = (kn/2)(VOV)²(1 + λVDS)

MOSFET equivalent of Early effect. Typically 0.01–0.1 V⁻¹. Longer channel → smaller λ → better current source. Defines output resistance ro = 1/(λID).

MOSFET is voltage-controlled (IG ≈ 0). BJT is current-controlled (IB drives IC). This is the fundamental architectural difference in circuit design.

gm is the most important small-signal parameter — it converts input voltage to output current. Higher gm = higher gain. For BJT: gm ∝ IC directly. For MOSFET: gm ∝ √ID, so doubling current only raises gm by √2.
COMPARE BJT vs MOSFET — Parameter by Parameter
Parameter BJT (NPN) MOSFET (NMOS)
Control mechanism Current (IB controls IC) Voltage (VGS controls ID)
Input impedance Moderate — rπ = β/gm (~kΩ) Extremely high — ~∞ at DC
Transconductance gm IC / VT — high at moderate IC √(2knID) — lower for same current
Output resistance ro VA / IC (VA = 50–200V) 1 / (λID) (λ = 0.01–0.1 V⁻¹)
Gain (gm·ro product) β·VA/VT — very high Lower — trades off with speed
I-V law Exponential: IC = IS·e^(VBE/VT) Square-law: ID = (kn/2)VOV²
Temperature behavior VBE drops ~2mV/°C; β changes VTH drops; μ decreases (→ safer)
Speed (fT) Very high — SiGe BJTs reach >300GHz High; limited by Cgs, Cgd
Noise Lower 1/f noise; good at RF Higher 1/f noise (flicker noise)
Integration density Larger — harder to scale Excellent — dominant in VLSI
Body effect None VSB ≠ 0 raises VTH (gmb)
Analog analog use Precision analog, low noise, RF CMOS analog, digital, mixed-signal
